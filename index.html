Maps

to reiterate: Arrays, Maps and Objects are Objects 

typeof Array(); // -> object
typeof new Map(); // -> object
typeof Object(); // -> object
with Arrays we use properties to keep track of multiple things, lists, queues, stacks, etc. With 

const a = ["these", "are", "multiple", "strings"];
// ask if a has a property "3"
a.hasOwnProperty("3") // -> true
a.hasOwnProperty(4) // -> false
with Objects we use properties to build things 

const hero = {
    "name": "atom",
    "powers": ["split", "fuse"]
}
but properties are strings, to keeping track of objects that have the same string representation is not possible 

const atom = {"name": "atom"};
const thunder = {"name": "thunder"};
const antimatter = {"name": "antimatter", evil: true };
const lightning = {"name": "lightning", evil: true };
const heroesToArchEnemy = {};
heroesToArchEnemy[atom] = antimatter;
heroesToArchEnemy[thunder] = lightning;
heroesToArchEnemy // -> "[object Object]": {name: 'lightning', evil: true}
